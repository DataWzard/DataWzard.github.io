# Jacob Stack portfolio

An editable, responsive portfolio for Jacob Stack. The site is built with Next.js and is configured for automatic deployment to GitHub Pages.

## Publish on GitHub Pages

Nothing is published automatically from this local folder. When you are ready:

1. Create a new public repository on GitHub.
2. Upload or push this project to the repository's main branch.
3. Open Settings > Pages and set Source to GitHub Actions.
4. The included workflow will build and publish the site.

The public/CNAME file keeps stacksanalytics.us as the custom domain. Update or remove it if you use a different domain.

## Edit locally

- Main content: app/page.tsx
- Visual design: app/globals.css
- Resume download: public/Jacob_Stack_Resume.pdf
